# TP: Sitio Web con HTML y CSS

Aca dejo el repositorio de mi trabajo practico para Programacion III de la UTN. Es una web estatica de 3 paginas donde aplique lo basico de HTML y CSS, siguiendo las reglas que pedia la consigna.

## Link al Sitio Web
El proyecto está publicado y accesible a través de GitHub Pages:
**🔗 [Haz clic aquí para ver el sitio web](https://lautarojover.github.io/TP-Sitio-Web-con-HTML-y-CSS//)**

##  Que tiene el proyecto?
Arme tres paginas que estan conectadas por un menu de navegacion:

1. **`index.html` (Inicio):**
   - Elegi como tema una intro basica a JavaScript y React.
   - Use las etiquetas semanticas de HTML5 (`header`, `nav`, `main`, `section`, `footer`).
   - Le agregue las dos imagenes que pedia el tp (una con ruta absoluta y otra local).

2. **`receta.html`:**
   - Puse una receta simple de un budin.
   - Aca meti la lista no ordenada (`<ul>`) para los ingredientes y la lista ordenada (`<ol>`) para los pasos de preparacion.

3. **`horario.html`:**
   - Arme una tabla con mi horario de cursada de la semana.
   - Use `colspan` y `rowspan` para acomodar los horarios y las materias que duran mas de una hora.

##  Como se armo?
* **HTML5:** Toda la estructura.
* **CSS:** Solo use colores, fuentes y tamaños de texto. Respetando el tp, no meti nada de margenes, padding ni modelos de caja. Todo re basico.
* **Git:** Fui subiendo el progreso de a poco con varios commits. Sin usar librerias raras ni frameworks.
