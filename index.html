JavaScript: El Lenguaje de la Web

¿Qué es JavaScript?

JavaScript (JS) es un lenguaje de programación que te permite añadir interactividad, dinamismo y funciones complejas a tus páginas web. Es el "cerebro" que controla el comportamiento del sitio.

Ejemplo Básico de Código

Aquí tienes cómo declarar una variable y mostrar un mensaje:

let nombreUsuario = "Lautaro";
console.log("¡Hola, " + nombreUsuario + "!");

React: Una Biblioteca de JavaScript

¿Qué es React?

React es una biblioteca de JavaScript de código abierto, desarrollada por Meta, que se utiliza para construir interfaces de usuario (UI). Es muy popular para crear aplicaciones web de una sola página (SPA).

React se basa en el concepto de componentes, que son piezas de código reutilizables y autónomas que definen cómo se ve y se comporta una parte de la pantalla.
